# 01_auth – Скриншотын бүтэц

Энэ хавтас нь аппын дэлгэцийн скриншотуудыг категориар ангилсан.

## Хавтаснууд

| Хавтас | Агуулга |
|--------|--------|
| **splash** | Апп нээлт — ANT INVEST GROW (splash screen) |
| **home** | Нүүр — ДАН таниулах, дижитал зээлийн таб, зээлийн CTA (3 дэлгэц) |
| **loan** | Зээлийн бүх урсгал: бүтээгдэхүүн, хүсэлт, төлөв, барьцаа, санал |
| **savings** | Хадгаламж (1–4: нүүр, дэлгэрэнгүй, орлого, хуулга), Дэлгэц 50–57 (12 дэлгэц) |
| **profile** | Профайл цэс, тохиргоо, нууц солих, хувийн мэдээлэл, банкны данс, гэр бүл, ажил эрхлэлт, холбогдох, салбар (12 дэлгэц) |
| **onboard** | Онбординг дэлгэцүүд (дэлгэц 9–11; жишээ: "Мөнгөнөөсөө Мөнгө ол") |
| **register** | Бүртгэл үүсгэх (дэлгэц 6), Үйлчилгээний нөхцөл (Зөвшөөрсөн) |
| **biometric (login)** | Нэвтрэх (дэлгэц 7), Нэвтрэсэн хэрэглэгч — хуруу, нүүр, ID (дэлгэц 8) |
| **resetpass** | Нууц үг сэргээх (регистр дугаар → утас баталгаажуулах → 6 оронтой код) |
| **dan_not_connected** | ДАН холболт хийгээгүй хэрэглэгчид харагдах дэлгэцүүд (нүүр, дан холболт, баталгаажуулалт) |
