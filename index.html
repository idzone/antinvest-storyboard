ANTInvest — Гол Storyboard

Аппын бүх дэлгэцийн скриншот ба хавтасны газрын зураг. Энэ хуудсыг хэрэглэгч болон backend хөгжүүлэгчид гол лавлагаа болгон ашиглана.
Flow диаграм (FlutterFlow маяг): STORYBOARD_FLOW.html

Энэ хуудас юу вэ? ANT Invest Club аппын дэлгэцүүдийг нэг дороос харах гол storyboard. Дэлгэц бүрт дугаар өгсөн (1, 2, 3, …) — зураач, хөгжүүлэгч «дэлгэц 12», «screen 23» гэж лавлагаа өгч болно. Зураг ямар хавтаст байгааг доорх ногоон шошгоноос харна.

Процессын урсгал (энэ дэлгэц → дараагийн дэлгэц)

Дэлгэцүүд процессын дарааллаар зураасаар холбогдсон. Доорх диаграм болон карт бүрийн доорх «Дараагийн: → Дэлгэц X»-аар холбоог харна.

Холбоос: Энэ storyboard нь GitHub Pages дээр hostлогдсон. Хэрэглэгчид энэ хуудсыг https://idzone.github.io/antinvest-storyboard/ хаягаар нээж үзүүлнэ.

Хэрхэн ашиглах

Хэрэглэгчид: Дэлгэц бүр ямар үйлдэл/хуудас болохыг зураг дээрээс харна. Цэснээс хэсгүүд рүү шууд үсрэнэ.
Backend хөгжүүлэгчид: Хавтас бүр нь нэг домэйн/flow (auth, loan, savings, profile). Доорх хүснэгтээс хавтас → юу харуулдаг → API/урсгал гэж ойлгоод, дэлгэцний нэр (файлын нэр)-ээр API эсвэл route-ийг таньж болно.
